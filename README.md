# Heart Disease Statistical Analysis

## 📌 Project Overview
The Heart Disease Statistical Analysis project performs a detailed analysis of the Cleveland Heart Disease dataset (303 records, 14 features). It focuses on identifying patterns and risk factors associated with heart disease using statistical methods and Python programming.

---

## 📊 Dataset
- Source: Cleveland Heart Disease Dataset  
- Records: 303 patients  
- Features: 14 medical attributes  

Links:  
https://www.kaggle.com/datasets/cherishmimi/heart-disease-cleveland-uci  
https://archive.ics.uci.edu/ml/machine-learning-databases/heart-disease/processed.cleveland.data  

---

## ⚙️ Features
- File handling with exception management  
- Data exploration (head, tail, info, filtering)  
- Missing value handling (including '?' values)  
- Feature engineering (RiskScore, BP-Cholesterol Ratio, AgeGroup)  
- Age-based analysis and comparisons  
- Dictionary-based data storage  
- Safe division handling (ZeroDivisionError)  
- Statistical calculations (mean, median, mode, min, max)  
- Report generation (heart_disease_analysis_report.txt)  

---

## 🛠️ Tech Stack
- Python  
- Pandas
